# Plataforma-de-estudio
Plataforma web académica basada en IA y arquitectura RAG.

## Tecnologías

- React
- TailwindCSS
- FastAPI
- PostgreSQL
- pgvector
- Gemini API
- JWT
- Bcrypt

## Funcionalidades

- Registro e inicio de sesión
- Carga de documentos PDF
- Chat inteligente mediante RAG
- Generación de resúmenes
- Flashcards automáticas
- Exámenes de práctica
- Métricas académicas
- Salas colaborativas

## Arquitectura

Cliente-Servidor Multi-Tenant

Frontend → React
Backend → FastAPI
Base de datos → PostgreSQL + pgvector
IA → Gemini API
